| ファイル | 用途 | 参照ページ |
| --- | --- | --- |
| `poster.pdf` | ポスター PDF | `/jsai2026/poster` |
| `slides.pdf` | 口頭発表スライド PDF | `/jsai2026/oral` |
| `paper.pdf`  | 論文 PDF（または `CommonResources.astro` で外部URLに変更可） | 両ページ |
